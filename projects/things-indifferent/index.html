My first web project.

Projects | 2013 | Links: Things indifferent.

This is my first web project that I developed for me and my friends in 2013. It’s simple and responsive. I used python with google app engine framework for backend and twitter bootstrap for the frontend. I was really fascinated by the new world of simple web development at that time.

In nutshell, this is a web constructor of courses + blog platform with a little bit of stoicism.

The supported features:

  • user registration
  • simple blog system with comments. (User can create posts but can’t delete anything)
  • simple to-do system. User can add new tasks, freeze them, finish (then some experience is gained) or fail. Privacy is allowed.

Source code on Github.
